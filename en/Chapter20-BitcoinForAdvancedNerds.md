# Bitcoin For Advanced Nerds



###### footnotes and quotes

I want this chapter to be an overview of advanced tools like trick PIN's, Seed XOR, shamir secret sharing, air-gapped computers, multisig, and miniscript. These things are fainating, but I also want to write about the tradeoffs that come with them. Just because hyou are a bitoicn expert does't mean their spouse is as advanced as they are. The idea is to write a short summation of each of these advanced tools you might find on various hardware wallets. An expert bitcoiner might know we only need the first 4 letters of each word to recover our seed, but an exzpert bitcoiner's spouse have no idea. Hopefully this can help some people navigate these more complex waters if they need to.
